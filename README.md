## ⚙️ Basic Setup and Usage

### Add a Website to Monitor

1. Click **Add New Monitor** or the plus (+) button.
2. Enter the website address (URL) you want to watch.
3. Choose the type of monitoring:
   - **Price tracking** to watch for price changes.
   - **Content monitoring** to track any change on the page.
4. Set how often you want openclaw-webwatcher to check the website. Options include every 10 minutes, hourly, or daily.
5. Save your monitor.

### Get Notifications

You will receive an alert whenever the software notices a change. This could be:

- A pop-up window inside the app
- A sound alert (can be turned off)
- An email (optional setup)
